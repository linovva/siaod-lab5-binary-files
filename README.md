# Практическая работа 5-1: Внешние структуры данных — текстовый и двоичный файлы  
**Дисциплина**: Структуры и алгоритмы обработки данных (СиАоД)  
**Задание 2, Вариант 1**

---

## Условие задания

Разработать программу управления **двоичным файлом с записями фиксированной длины**.  
**Структура записи** — *читательский абонемент*:
- **Номер читательского билета** — целое пятизначное число (**ключ**)
- **ФИО** — строка до 50 символов
- **Адрес** — строка до 50 символов

### Обязательные операции
- Преобразование тестовых данных из текстового файла в двоичный  
- Сохранение данных двоичного файла в текстовом виде (с возможностью восстановления)  
- Вывод всех записей двоичного файла  
- Доступ к записи по порядковому номеру (прямой доступ)  
- Удаление записи по ключу путём замены на последнюю запись  

### Дополнительные операции
- Поиск записи с заданным значением ключа  
- Удаление найденного значения
